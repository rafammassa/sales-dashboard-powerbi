# 📊 Análise de Vendas | Power BI

Dashboard desenvolvido com foco em análise de faturamento, lucro e performance comercial.

## 🎯 Objetivo
Transformar dados de vendas em insights claros para apoio à tomada de decisão.

## 📌 Métricas analisadas
- Faturamento
- Lucro
- Margem
- Quantidade de vendas

## 📊 Análises realizadas
- Evolução de faturamento ao longo do tempo
- Performance por região
- Análise por categoria e subcategoria de produtos

## 🛠️ Ferramentas utilizadas
- Power BI
- DAX

## 📁 Dataset
Dados baseados em dataset público de vendas.

---

Projeto desenvolvido como parte da construção de portfólio em análise de dados.
